# 🏸 羽毛球MBTI人格测试网站

一个交互式的羽毛球人格测试网站，通过9道精心设计的问题，帮助用户了解自己在球场上的真实人格类型。

## 📋 项目简介

这个网站通过测试用户的三个维度来判定其羽毛球人格类型：
- **核心动力维度**：你为什么来打球？（竞争型 vs 玩乐型）
- **技术风格维度**：你在场上怎么打？（攻击型 vs 防守型）
- **沟通模式维度**：你在场上怎么交流？（表达型 vs 安静型）

最终会生成8种不同的人格类型组合。

## ✨ 功能特点

- ✅ 9道精心设计的测试题目
- ✅ 美观现代的UI设计
- ✅ 流畅的交互动画效果
- ✅ 完整的人格类型判定逻辑
- ✅ 详细的结果展示和描述
- ✅ 结果分享功能
- ✅ 响应式设计，支持手机和电脑访问
- ✅ 无需后端，纯前端实现

## 🎯 人格类型说明

网站会根据用户的答案判定出以下8种人格类型之一：

1. **CAE** - 竞争型-攻击型-表达型：充满激情的竞技者
2. **CAD** - 竞争型-攻击型-安静型：沉默杀手
3. **CDE** - 竞争型-防守型-表达型：智慧型指挥官
4. **CDQ** - 竞争型-防守型-安静型：冷静分析家
5. **PAE** - 玩乐型-攻击型-表达型：活力四射者
6. **PAD** - 玩乐型-攻击型-安静型：自由主义者
7. **PDE** - 玩乐型-防守型-表达型：社交型玩家
8. **PDQ** - 玩乐型-防守型-安静型：佛系玩家

## 📝 判定逻辑

- 每个维度有3道题目
- 如果某个类型的得分 ≥ 2，则判定为该类型
- 最终人格类型 = 核心动力类型 + 技术风格类型 + 沟通模式类型
